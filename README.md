## 📌 About

**Saffron & Spice** is a full-featured recipe website where users can:

- 🔍 Search and filter recipes by name, ingredient, or cuisine
- ❤️ Save their favourite recipes
- 🛒 Add recipes to a cart and checkout
- 📖 View full recipe details with nutrition info
- ✉️ Contact the team using a validated form

Built using **only 3 files** — no frameworks, no npm, no installation needed.

<br>

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🛒 Cart System | Add recipes, change quantity, live total, checkout |
| ❤️ Favorites | Save recipes with heart button, view on Favorites page |
| 🔍 Search & Filter | Live search + 8 cuisine filter tabs |
| 🪟 Recipe Modal | Full popup with photo, nutrition, and ingredients |
| 📱 Responsive | Works on mobile, tablet, and desktop |
| ✅ Form Validation | 5-field contact form with real-time error checking |
| 🌀 Loading Screen | Splash screen on page load |
| ✨ Blur Animation | Smooth blur fade-in effect when page loads |

## 🚀 How to Run

No installation needed! Just follow these steps:

```
1. Download or clone this project
2. Open the folder
3. Double-click index.html
4. It opens in your browser — done! ✅
```

> Make sure you have an internet connection for the food photos to load (they come from Unsplash).
